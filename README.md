# 🛒 E-Commerce React App ( My-OnlineShop )

A modern React-based E-Commerce web application with cart, wishlist, product listing, product details, and authentication structure using Context API and Mock API.

---

# 🚀 Features

## 📦 Product Features
- Product listing from API
- Product details page
- Category filtering
- Search functionality (live filter)
- Related products section

---

## 🛒 Cart System
- Add to cart
- Increase / decrease quantity
- Remove items
- Total price calculation
- Persistent cart (localStorage)

---

## ❤️ Wishlist System
- Add / remove wishlist items
- Wishlist page
- Persistent wishlist (localStorage)

---

## 👤 User System
- Login / Register pages
- Protected routes
- User dashboard support

---

## 🎨 UI Features
- Responsive design
- Product grid layout
- Clean modern UI
- Sticky layout structure
- Image fallback handling

---

# ⚙️ Tech Stack

- React.js ⚛️
- React Router DOM 🔗
- Context API 🧠
- Axios 🌐
- MockAPI (Backend) 🌍
- CSS (Custom Styling) 🎨

---

# ⚙️ Installation & Setup

### 1. Clone repository

```bash
git clone https://github.com/chavannandini28/my-onlineshop.git
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start development server

```bash
npm run dev
```
